# 🎨 Repositório de Estudos — Web Responsivo, UI e UX

Este repositório foi criado com o objetivo de reunir **práticas, testes e projetos de aprendizado** nas áreas de desenvolvimento web responsivo, design de interfaces e experiência do usuário.

## 🚀 Objetivo

Explorar e evoluir habilidades em:

- 📱 **Web Responsivo**  
  Criação de layouts adaptáveis para diferentes tamanhos de tela e dispositivos.

- 🎯 **UI (User Interface)**  
  Desenvolvimento de interfaces visuais organizadas, acessíveis e agradáveis.

- 🧠 **UX (User Experience)**  
  Foco na usabilidade, navegação intuitiva e melhoria da experiência do usuário.

## 🛠️ O que você pode encontrar aqui

- Estruturas HTML semânticas  
- Estilização com CSS moderno  
- Layouts responsivos com Flexbox e Grid  
- Protótipos de interfaces  
- Pequenos projetos experimentais  
- Exercícios de melhoria visual e usabilidade  

## 📚 Propósito do repositório

Este projeto não tem fins comerciais.  
Ele serve como um **ambiente de prática contínua**, evolução técnica e organização do aprendizado.

## ✍️ Autor

Desenvolvido como parte do processo de aprendizado em desenvolvimento web e design de interfaces.
